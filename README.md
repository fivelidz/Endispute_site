## Deployment

### Local Development
1. Open `index.html` in a modern web browser
2. For full functionality, serve from a local web server:
   ```bash
   python -m http.server 8000
   # or
   npx serve .
   ```

- Ensure HTTPS is enabled

- Set up CDN for global performance
- Monitor Core Web Vitals
